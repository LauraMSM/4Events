Este archivo es para editar la información del usuario en la base de datos. Primero se incluye la conexión a la base de datos con la función get\+DB(). Luego, si se ha recibido la variable \textquotesingle{}edit\textquotesingle{} en el método POST, se actualiza la información del usuario en la base de datos.
